# Flow-app
VueJS + Quasar + nodeJS pwa
Progressive Web Application for Company Management (CRM)
# How to run:
 - Open a terminal in the main directory
 - Run XAMPP or any other DBMS for SQL databases
 - In terminal:
 - $cd back-end
 - $npm install
 - $nodemon/npm start
 - Open another terminal
 - $cd front-end
 - $npm install
 - $quasar dev
